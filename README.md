# my to-do app
Simple personal to-do list management app.
## Features
 - Add to-do items to the pending list
 - Mark the to-do items as Done
 - Revert the to-do items back to pending
 - Remove to-do items
 - Store to-do item on database, keep your to-do data!
 - Store your personal to-do items separately from other users
## Database
Firebase Realtime Database
## React.js related
 - `useState`, `useContext` -- managing to-do state data, CRUD with firebase database
 - `useReducer` -- Http state helper, loading spinner and error handler
 - `localStorage` -- Store self-generated `userId` to assign to the items
